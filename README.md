# Lia - IA parceira proativa

Lia é uma IA estilo Jarvis:
- Proativa: age sem esperar comandos.
- Autônoma: planeja e toma decisões.
- Multimodal: pesquisa, acessa dispositivos e câmera.

Objetivo:
- Criar um agente de IA que entiende minhas rotinas,
- sugira ações,
- e execute tarefas no meu computador.
